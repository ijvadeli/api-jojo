# JoJo API

This is an API for pulling random quotes, gifs and memes of the series JoJo's Bizarre Adventure.

View the Documentation here: [Link]()

## Features

This API can:
- Random quotes
- Random gifs/memes
- Random images/art
- Option for audio files

## Tech Stack

- Node.JS
- Express
